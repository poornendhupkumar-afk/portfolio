01 — About

A bit about me

I'm a second-year B.Tech Computer Science student at Model Engineering College, Thrikkakara. My core interest is applied machine learning — not just training models, but getting them into a state where someone other than me can actually use them.

That's pulled me into a few adjacent areas: Python tooling and automation to make experiments repeatable, cybersecurity to understand how the systems I build can fail, and just enough web development to ship a model behind a real interface instead of a notebook.

I learn best by finishing things. Every project below started as a way to get unstuck on a specific problem, not as a portfolio checkbox.
